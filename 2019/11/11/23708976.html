Docker命令笔记

docker镜像相关命令

  • docker search [options] name：搜索镜像

    docker search nginx

  • docker [image] pull name[:tag]：拉取镜像，name是镜像仓库名，tag是镜像标签（一般表示版本信息），如果不指定tag，默认会选择latest标签，拉去仓库中最新版本镜像；如果拉去的不是默认仓库的镜像，可以在name前面写上仓库完整路径。

    docker image pull nginx:1.17：拉取nginx 1.17版本镜像
    docker pull hub.c.163.com/public/nginx:1.17：从网易蜂巢的镜像源拉取nginx

  • docker image ls/docker images：显示本地所有镜像

    docker ls
    docker images

  • docker image tag name:tag newname:newtag：为本地镜像添加新的tag

    docker image nginx:1.17 mynginx:new

  • docker image inspect name:tag：查看本地镜像详细信息

    docker image inspect nginx:1.17

  • docker image rm name:tag：删除本地镜像

    docker image rm nginx:1.17

docker容器相关命令

  • docker [container] create [options] name[:tag]：创建容器，创建的容器处于停止状态，需要用docker container start来启动

    docker create -it nginx:1.17
    常用选项：
    -i，–interactive=true|false：保持标准输入打开，默认为否
    -d，–detach=true|false：是否在后台运行容器，默认为否
    -t，–tty=true|false：是否分配一个伪终端，默认为否
    –rm=true|false：容器退出后，是否自动删除，不能与-d同时使用
    –name：指定容器的别名

  • docker [container] start nginx：启动容器

    docker container start nginx

  • docker ps：查看容器

    docker ps：查看运行中的容器
    docker ps -a：查看所有容器

  • docker [container] run [options] nginx[:tag]：创建并运行容器，等价于先执行container create，再执行container start。

    docker container run -d –name nginx

  • docker exec [options] name bash：进入容器内部

    docker exec -it nginx /bin/bash
